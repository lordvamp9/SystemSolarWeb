# Space - Solar System Visualization

This project is a 3D visualization of the Solar System built using HTML, CSS, and JavaScript with the Three.js library. It features a glowing Sun, rotating planets with orbits, a starry background, and a nebula effect.

## Controls

Explore the universe using the following controls:

- **W, A, S, D**: Move the camera (Forward, Backward, Left, Right).
- **Mouse**: Click and drag to look around (Orbit).
- **Mouse Wheel**: Zoom In / Zoom Out.
- **Shift**: Hold to move faster.
